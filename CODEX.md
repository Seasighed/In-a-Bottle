# Codex Project Context

Canonical Codex guidance for this repo lives here:

- `Dev/AI Engine/AI Agent Standards.md`

Working preferences for this project:

- Compile after meaningful code changes when practical. Verification is preferred.
- Do not guess token counts. If the Codex runtime does not expose them, record them as unavailable.
- At the end of each Codex work session, append or create `Log/YYYY-MM-DD.md`.
- Daily logs should include work completed, files touched, verification performed, blockers, and next steps.
- Keep contributions attributable. Logs should clearly distinguish Codex work from user work when known.
